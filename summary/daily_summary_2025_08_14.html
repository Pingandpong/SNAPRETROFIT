오늘의 주요 작업: 다국어 지원 (i18n) 기능 구현

오늘은 i18next와 react-i18next 라이브러리를 사용하여 앱 전체에 다국어 지원 기능을 구현했습니다. 사용자가 설정 화면에서 언어를 선택하면 UI 언어가 실시간으로 변경됩니다.

  • 라이브러리 및 설정:
    • i18next, react-i18next 라이브러리를 설치하고, package.json에 의존성을 추가했습니다.
    • src/services/i18n.ts 설정 파일을 생성하여 i18next 인스턴스를 초기화하고, 지원할 언어 리소스를 구성했습니다.
    • App.tsx에서 설정 파일을 임포트하여 앱 시작 시 다국어 지원 기능이 활성화되도록 했습니다.
  • 번역 리소스 추가:
    • src/locales 디렉토리를 생성하고, 총 4개 국어에 대한 번역 파일을 추가했습니다: 한국어(ko), 영어(en), 일본어(ja), 스페인어(es).
    • 각 파일에는 화면에 표시될 모든 텍스트에 대한 번역 키와 값을 정의했습니다.
  • UI 적용 및 언어 변경 기능 구현:
    • HomeScreen.tsx: 기존의 하드코딩된 텍스트를 모두 useTranslation hook의 t('key') 함수를 사용하도록 리팩토링했습니다.
    • SettingsScreen.tsx: 언어 변경 UI를 추가했습니다. 사용자는 4개의 언어 버튼 중 하나를 선택하여 앱의 언어를 동적으로 변경할 수 있습니다.

이번 작업을 통해 앱의 글로벌 확장성을 위한 기반을 마련했습니다. 새로운 언어를 추가하는 작업 또한 매우 간단해졌습니다.
