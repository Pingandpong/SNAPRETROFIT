# App Base

이 저장소는 Expo와 React Native로 구축된 모바일 앱 베이스입니다. Firebase 설정, 다국어(i18n), 테마 전환 등 대부분의 앱에서 공통으로 필요한 기능을 미리 포함하고 있습니다.

## 설치
```bash
npm install
```

## 실행
```bash
npm start       # 개발 서버 시작
npm run android # 안드로이드 실행
npm run ios     # iOS 실행
```

## 테스트
```bash
npm test
```

## 환경 변수
필요한 환경 변수는 `.env` 파일로 관리되며, 예시는 `.env.example`을 참고하세요.
