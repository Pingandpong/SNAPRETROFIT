MVP 추천안: SnapRetrofit — Photo-first Home Energy Auditor & Incentive RAG Planner

핵심 요약
- 문제/기회: 에너지 요금 급등 + 정부/지자체 리베이트 확대. 집주인·소형 임대주는 “무엇부터, 얼마나 절약되는지, 보조금은 어떻게 받는지”를 모름. 전문가를 부르기 전, 사진만으로 우선순위/ROI/보조금 루트를 뽑아주면 바로 지갑이 열린다.
- 와우 포인트: 집 구조(다락문, 창호, 실외기) 몇 장 찍으면, 누수 지점 하이라이트 + 연간 절감액 + 투자회수기간 + 지역 리베이트 체크리스트가 2분 내 생성. 사진 기반 ‘준-전문가급’ 리포트.
- Money-Making AI Feature: Vision(사진→결함·기기 식별) + RAG(지역 인센티브·규정·단가 문서 벡터 검색) 결합으로 “우선순위 액션 + ROI + 보조금 링크”를 자동 생성. 유저는 PDF 리포트/IAP를 결제하고, 추가로 ‘검증 견적(Contractor Verified)’ 유료 업셀에 진입.

왜 유료 결제할까
- 직접 조사 대비 시간 절약: 리베이트/규정/공사 우선순위 찾기 3–5시간 → 5분. 스트레스 제거.
- 금전적 임팩트: 연간 절감액과 보조금 합산으로 의사결정이 즉시 가능. 1건만 실행해도 수십~수백 달러 절약.
- 외부 커뮤니케이션: 계약업체에 바로 보내는 ‘사진 주석+범위+예상비용’ 리포트로 견적 왕복 횟수 50%↓. 빠른 착수로 체감 가치가 크다.

MVP 범위(7일)
- Day 1–2: 데이터·AI 파이프라인
  - Supabase Auth/DB/Storage/Vector 셋업, RLS로 프라이버시 기본기.
  - 한 개 론치 지역 리베이트/유틸 문서 스냅샷 후 임베딩(벡터 저장).
  - OpenAI Vision으로 3가지 감지: 다락/단열 힌트, 창호 타입(single/double-pane), 실외기 모델·연식 스티커.
- Day 3–4: 카메라 우선 UX + RAG 추천
  - Expo Router + NativeWind로 촬영 가이드(각도/거리/조명)·품질체크(해상도/블러)·업로드.
  - RAG로 우선 조치 2–3개, 비용범위, 연간절감액, Payback, 출처 링크 생성.
- Day 5: 수익화 요소
  - 한 페이지 PDF(사진 주석, 우선순위, ROI, 리베이트 체크리스트) 생성 → IAP/구독으로 잠금.
- Day 6: 신뢰·비용 관리
  - 보수적 추정 모드, 면책 문구, 사진 보존기간 설정(자동 삭제 옵션).
- Day 7: 업셀/시장 진입
  - ‘검증 견적’ 워크플로(유료): 리포트를 지역 시공사에 전달해 확인·보정. 리드 과금/커미션 구조 테스트.

수익 모델(초기 가설)
- Freemium → Premium 구독: $4.99/월 또는 $49/년(무제한 상세 스캔+리베이트 RAG+월 3회 PDF).
- IAP: 프로 리포트 $9.99–$14.99, ‘검증 견적’ $29.99.
- B2B 보조 수익: 시공사 리드 과금($50–$150/성사) 또는 저가 구독.

기술 스택과 “개발 시간 절약” 포인트
- Frontend: Expo Router + NativeWind → 카메라 플로우/온보딩/페이월을 1명/7일에 구현 가능.
- Backend: Supabase Auth/DB/Storage + RLS → 로그인/권한/사진보관을 서버 없이 신속 구축.
- Supabase Vector: 지역 리베이트·규정 문서 임베딩/검색을 관리형으로 처리 → 자체 검색 인프라 구축 시간 절감(수일 → 수시간).
- OpenAI Vision + GPT-4o: 커스텀 CV 모델 학습 없이도 결함 감지/스티커 OCR/구조화가 가능 → 모델 파이프라인 구축을 크게 단축.
- 결과: 풀스택 MVP를 7일 내 엔드투엔드로 연결, 이후 확장(지역 추가/모델 정교화)에 리소스 집중.

리스크와 완화
- 지역 데이터 최신성: 1개 지역부터 시작, 벡터 코퍼스 소규모 유지, 주간 크롤/갱신.
- 비전 정확도: 촬영 가이드·품질체크로 입력 개선, 보수적 판단(정밀도 우선), 의심 구간은 ‘검증 견적’ 업셀로 전환.
- 책임 이슈: 추정 범위·면책 고지, 출처 링크, 보수적 하한치 제시.

지표(초기 2주 베타)
- 스캔→분석 성공률 ≥85%
- 리포트 결제 전환 ≥3%
- ‘검증 견적’ 업셀 ≥1%
- 스캔당 토큰 비용 <$0.30(캐시/요약로직 적용)

왜 지금, 왜 이거
- 에너지·보조금은 2025년에도 뜨거운 주제. “사진 → 돈이 되는 액션플랜”은 즉시성·금전효과가 커서 지갑을 연다.
- AI 없이는 불가능: 비전 탐지 + 지역 규정/리베이트 RAG 결합이 가치의 90%. 단순 CRUD 아님.
