# PrivateDiary — 온디바이스 AI 일기 & 회고

## ✨ 앱 컨셉 / 핵심 목표
민감한 일기를 **네트워크 없이** 기록·요약·감정분석까지 **모두 기기 내에서 처리**하는 오프라인 우선 모바일 앱.  
**프라이버시 100% + 즉각 반응 속도**로 빠른 일상 기록과 회고 습관화에 집중합니다.

---

## 🧱 간단 기술 스택

### 📱 클라이언트
- **React Native + Expo**
  - 빠른 크로스플랫폼 개발
  - OTA 업데이트 가능

### ☁️ 최소 백엔드
- **Firebase (무료 중심)**
  - Auth(익명/이메일)
  - FCM(푸시)
  - Remote Config(플래그 제어)
- **Cloud Run (선택)**
  - 장문 생성 등 고연산 작업만 오프로딩
  - 스케일-투-제로로 비용 거의 0원 수준 유지

### 🤖 온디바이스 AI
- **TFLite / ONNX**  
  - 한국어 경량 LLM(3–7B, 양자화)  
  - 온디바이스 감정 태깅  
  - 오프라인 ASR(음성 → 텍스트)

### 🔐 데이터 저장
- 로컬 암호화 저장소 (Encrypted Storage 또는 Realm)
- 기기 내 인덱싱 기반 자연어 검색

---

## 🚀 MVP(핵심 기능)

1. **텍스트 일기 작성**
   - 온디바이스 요약
   - 온디바이스 감정 태깅

2. **오프라인 음성 기록**
   - 녹음 → 온디바이스 음성 인식 → 자동 텍스트 변환

3. **자연어 검색**
   - “지난달 기분 나빴던 날” 같은 질의 검색

4. **간단 UI**
   - 작성 / 보관함 / 검색 / 설정

5. **일일 회고 푸시 알림**

6. **모든 데이터 로컬 저장**
   - 추후 선택적 백업 옵션 추가 예정

## ⚠️ 잠재 리스크 & 유의사항

### 1. 한국어 온디바이스 모델 품질 편차
- 초기엔 **간단 요약/감정**으로 범위를 제한  
- 장문 요약 필요 시 Cloud Run + OpenAI로 선택적 전환

### 2. 배터리 & 메모리 이슈
- 기기 성능 감지  
- ‘저전력 / 표준 / 고품질’ 모드 제공

### 3. 앱 번들 크기 증가
- 모델은 **앱 실행 후 다운로드** 방식으로 분리

### 4. 개인정보 및 심사 이슈
- “데이터는 기본적으로 오프라인 저장”을 UI·정책에 명확히 표기  
- 사용자 동의 없는 업로드 없음
