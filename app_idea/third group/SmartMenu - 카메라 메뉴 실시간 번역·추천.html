SmartMenu - 카메라 메뉴 실시간 번역·추천

4–6주 내 MVP 출시를 목표로 한 1인 개발자 맞춤 설계

앱 개요 및 핵심 기능

  • 핵심 JTBD: “메뉴 사진을 찍으면 바로 번역·추천 받아서 주문을 쉽게 하고 싶다.”
  • 최소 기능:
    • 카메라 실시간 메뉴 텍스트 인식(OCR)
    • 즉시 번역 (다국어 → 한국어/영어)
    • 대표 이미지/추천 메뉴 표시
    • 즐겨찾기/최근 번역 내역 저장

기술 스택

  • React Native + Expo (EAS 빌드, 네이티브 OCR 모듈 포함)
  • UI: Gluestack UI
  • OCR: Google ML Kit
  • 번역: OpenAI GPT-4o-mini API
  • Firebase: Auth, Firestore, Analytics

수익 모델

  • 광고: 무료 번역 시 전면/배너 광고
  • 구독: 광고 제거, 번역 속도 향상, 무제한 번역
  • IAP: 특정 언어팩 구매

시장성 (왜 지금?)

  • 관광 회복 및 외식 트렌드 확산 → 외국인·외국어 메뉴 노출 증가
  • 영어·중국어·일본어 메뉴 이해 어려움 → 즉시 번역 수요 상승
  • 한국 외식업체의 외국인 고객 유치 증가
