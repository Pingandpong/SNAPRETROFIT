ReceiptSnap Lite - 영수증 스캔 지출 요약

RN/Expo + Firebase + Gluestack UI · 한/영 동시 타겟

앱 개요 및 핵심 기능

  • 핵심 JTBD: “영수증 사진 한 장으로 합계와 카테고리를 즉시 보고, 월 지출을 대략 파악하고 싶다.”
  • 최소 기능:
    1. 카메라/갤러리에서 영수증 선택 → ML Kit OCR 인식.
    2. 총액/날짜/상호 추출, 자동 카테고리(식비/교통 등).
    3. 월별 합계 카드 + 최근 항목 리스트.
    4. CSV 내보내기(이메일/구글시트).
    5. (옵션) 간단 예산 목표/알림.

기술 스택

  • RN/Expo: expo-image-picker, react-native-vision-camera(선택)
  • OCR: ML Kit Text Recognition v2(네이티브 모듈)
  • 분류/정규화: OpenAI(gpt-4o-mini)로 상호→카테고리 매핑
  • UI: Gluestack UI
  • DB/분석: Firebase Auth + Firestore + Analytics, Remote Config
  • MCP: 영수증 포맷별 파서/카테고리 룰 자동화

수익 모델

  • 무료+광고(리스트/통계 화면)
  • 구독(월 $2.99): 광고 제거, 무제한 내보내기, 커스텀 카테고리, 다중 통화

시장성 (왜 지금?)

  • 한국: 소액 결제 빈도↑, 가계부 진입장벽(직접 입력) 때문에 이탈↑.
  • 글로벌: 현금/종이 영수증 여전, 초간단 scan → total 니즈 상시.
