✅ 최종 추천
1) Snap2Schedule — 스샷 일정 추출·캘린더 등록 (1위)

왜 이걸? 한국/글로벌 모두 “공지 스샷 → 일정 놓침” 고통이 즉각적이고 반복적. 온디바이스 OCR + 경량 LLM 파싱 조합이면 1인도 1–2주 내 MVP 가능. 결과물(확인 카드/ICS)로 첫 세션 가치가 확실. 

빠른 7일 실행

D1–2: 스샷 선택→제목/날짜·시간/장소 추출→확인 카드에서 수정→로컬 알림 저장

D3–5: KR/EN 자동감지·ICS 내보내기·반복 규칙(간단)

D6–7: 온보딩/광고 유무·알림 프리셋 A/B, 핵심 전환 추적

핵심 지표/Kill

Create(일정 생성) ≥ 50%, Edit(수정) ≤ 30%, D1 ≥ 20%

150 installs 후 Create < 30%면 피벗/중단

월간 비용(Cost to Serve)

ML Kit OCR 온디바이스 중심 → 0

LLM 파싱(짧은 JSON 구조화) 500–1,000건/월 가정 → $5–20 수준(소규모)

2) QuickNoteCard — 한 줄 메모 → 예쁜 카드 공유 (차선)

왜 이걸? “메신저·스토리용 문장”을 즉시 카드화하는 초경량 JTBD. 번역·문구 보정은 짧은 호출로 제어 가능 → 초기 비용 낮음, 광고/IAP 실험 용이. 

빠른 7일 실행

D1–2: 입력→KR/EN 카드 2테마→공유

D3–5: 톤 프리셋(깔끔/재치)·최근 5개 저장

D6–7: 광고 위치 A/B·IAP(테마팩) 실험

핵심 지표/Kill

Generate ≥ 60%, Copy/Share ≥ 40%

150 installs 후 Copy < 20%면 피벗

월간 비용(Cost to Serve)

짧은 번역/보정 호출 500–1,000건/월 → $3–10 수준
