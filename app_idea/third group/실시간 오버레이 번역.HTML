Android 실시간 오버레이 번역 MVP 설계서

다른 앱 화면의 텍스트를 인식→번역→원문 위 오버레이로 표시하는 1인 개발자용 MVP 설계

앱 개요 및 핵심 기능

  • 목표 & 비전: 앱 전환 없이 보이는 텍스트를 즉시 이해. 1차 타겟: 카톡/WhatsApp/인스타 DM/웹뷰 기반 채팅 사용자(한국+글로벌).
  • 사용자 스토리 & JTBD: 외국어 메시지를 즉시 이해하고 싶고, 번역문이 원문 위치에 맞춰 겹쳐보이길 원함. 배터리/프라이버시 우려 해소.
  • 기능 범위(MVP): 실시간 화면 캡처, 텍스트 인식(OCR), 번역(OpenAI GPT API), 오버레이 UI, 알림 번역(옵션), 설정, 히스토리.

기술 스택

  • React Native + Expo (EAS 빌드로 네이티브 모듈 포함)
  • UI: Gluestack UI
  • OCR: ML Kit Text Recognition v2
  • 번역: OpenAI API (gpt-4o-mini 추천)
  • 상태: Zustand/Recoil, 분석: Firebase Analytics + Crashlytics
  • 원격 구성: Firebase Remote Config, 광고: Google Mobile Ads
  • 아키텍처 개요: RN은 제어·설정 UI와 상태표시 담당, 네이티브 Service는 프레임 파이프라인/오버레이/성능 튜닝 담당.
  • 네이티브 인터페이스: TypeScript (RN Bridge) 및 Kotlin (Android) 코드 스니펫 제공.
  • Firebase 스키마: 사용자, 이벤트, 실험 데이터 구조 예시.

수익 모델

  • 무료+광고: 히스토리/설정 화면 배너(오버레이엔 미표시).
  • 구독: 광고 제거, 고속 번역, 히스토리 무제한, 사용자 사전.
  • 가격: 월 $3–5 (현지화).

시장성 (왜 지금?)

  • 성공지표: 평균 지연 ≤ 2초, 세션당 번역 ≥ 10회, D1 유지 ≥ 20%.
  • 7일 검증 플랜: 내부 테스트 후 커뮤니티 소프트런칭 (DAU≥20%, 세션당 번역≥10, 평균 지연≤2초 목표).
  • ASO/마케팅:
    • KR 키워드: 번역 오버레이, 카톡 번역, 화면 번역.
    • EN 키워드: screen translator, chat overlay translate.
    • 크리에이티브 메시지: “다른 앱 위에 바로 번역”, “터치 없이 자동 번역”, “Instant translation on top of any app”.
