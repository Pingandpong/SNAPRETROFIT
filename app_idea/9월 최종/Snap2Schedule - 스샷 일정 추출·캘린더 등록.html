Snap2Schedule - 스샷 일정 추출·캘린더 등록

GEMINI.md 베이스 · RN/Expo + Firebase + Gluestack UI · 한/영 동시 타겟

앱 개요 및 핵심 기능

  • 핵심 JTBD: “공지/메시지를 스샷·붙여넣기만 하면 제목·날짜·시간·장소가 자동 인식되고, 1탭으로 알림까지 걸어 저장하고 싶다.”
  • 최소 기능:
    1. 입력: 스크린샷/갤러리 선택 또는 텍스트 붙여넣기(공유 시트 지원).
    2. 자동 추출: 제목/날짜·시간(타임존)/장소 파싱, 언어 자동감지(KR/EN).
    3. 확인 카드: 시작/종료/알림 시각/메모 빠른 수정.
    4. 저장 & 알림: 로컬 알림(expo-notifications), ICS 파일 내보내기.
    5. (옵션) 반복 규칙 감지(매주/격주 등 간단 패턴).

기술 스택

  • Expo: expo-image-picker, (필요시) react-native-vision-camera, expo-notifications, expo-sharing.
  • OCR: Google ML Kit Text Recognition v2 (네이티브 브리지).
  • 파싱: OpenAI gpt-4o-mini로 구조화(JSON) → 정규식/룰로 보정(chrono-like).
  • Firebase: Auth(익명), Firestore(히스토리/실험 플래그), Functions(파싱), Analytics, Remote Config(A/B).
  • UI (Gluestack UI): Card(추출 결과), Input/Select(시간/알림), Badge(언어), Button(저장/공유), Sheet(옵션).
  • MCP: 파싱 프롬프트 템플릿 자동 생성·튜닝(혼합 언어/표현 대응), ICS 생성 스크립트, ASO 문구 다국어화.

수익 모델

  • 무료 + 광고: 히스토리 화면 하단 배너(생성·승인 화면엔 미노출).
  • 구독(월 $2.49): 광고 제거, 반복 규칙/다중 알림, 클라우드 백업 & 동기화, 템플릿(회의/수업/택배).
  • IAP: 1회 프리미엄 팩(반복·다중 알림 일시 해금).

시장성 (왜 지금?)

  • 한국: 카톡·밴드·학원/회사 공지를 스샷으로 공유하는 문화 → 일정 누락 잦음.
  • 글로벌: DM·이메일·포스터 사진에서 날짜/시간/장소 추출해 캘린더로 저장하려는 상시 니즈.
  • 기술: 온디바이스 OCR + 경량 LLM 파싱으로 1인 개발도 빠른 체감 가치 제공 가능.
