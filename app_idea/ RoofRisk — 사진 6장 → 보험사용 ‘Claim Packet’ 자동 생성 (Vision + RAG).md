최종 추천: RoofRisk — 사진 6장 → 보험사용 ‘Claim Packet’ 자동 생성 (Vision + RAG)

- Money-Making AI Feature (와우 포인트)
  - 원터치 Claim Packet Generator: 사용자가 지붕 사진 5–8장을 찍으면 OpenAI Vision이 손상 유형/영역을 자동 감지·주석 처리하고, Supabase Vector에 쌓인 보험사별 템플릿·FAQ·로컬 코드 스니펫을 RAG로 매칭해 ‘보험사 문구’로 정리된 요약·체크리스트·권장 다음 단계까지 생성. 결과물을 고해상도 PDF/ZIP으로 즉시 내보내기(유료).
  - 왜 돈을 낼까: 
    - 홈오너: 클레임 자료 준비에 보통 2–3시간(사진 정리, 문구 작성, 업로드)을 절약. 평균 3~5일 빠른 초기 심사로 스트레스·현금흐름 개선.
    - 소규모 시공사: 현장 보고서·견적 문서화에 건당 30–60분 절약, 전문성 있는 문서로 전환율 상승(+10~20% 체감).

- 타깃과 문제
  - 홈오너: “이 정도 손상으로 청구가 될까?” → 불확실성/행정 피로.
  - 시공사: “현장에서 바로 보험사·고객 설득용 자료?” → 보고서 제작이 느리고 형식이 제각각.
  - Solution: 사진→주석→보험사 맞춤 요약→PDF 내보내기까지 1분 내 자동화.

- 기술 스택이 시간을 어떻게 아껴주나
  - Supabase Auth/Storage: 회원/세션/이미지 업로드를 바로 사용 → 인증/스토리지 백엔드 구현 시간을 2~3일 절감.
  - Supabase Vector: 보험사별 템플릿·FAQ를 임베딩으로 관리 → 검색/랭킹/버전관리를 서버less로 처리, RAG 파이프라인 구축 2일 내 완성.
  - OpenAI Vision + GPT: 커스텀 CV 모델 학습 없이도 손상 감지/설명/구조화 응답 가능 → 모델 학습·MLOps 수주 단축.
  - 결과: 1인 개발자 기준 7일 내 카메라→분석→RAG 요약→유료 내보내기 루프 완성.

- 7일 MVP 범위(핵심 일정)
  - Day 1–2: Supabase 프로젝트(Auth/DB/Storage/Vector) + OpenAI Vision/RAG 엔드투엔드 테스트(사진→라벨→RAG 요약).
  - Day 3–4: 카메라 가이드샷·온디바이스 QC(흐림/각도) + 주석 UI + 보험사 맞춤 요약 카드/트리아지 라벨.
  - Day 5: Claim Packet PDF 템플릿(표지/주석사진/체크리스트/라인아이템) + 유료 내보내기(paywall) 구현.
  - Day 6: 결제/권한(Stripe→Supabase Entitlement) + 온보딩(주소/보험사 선택) + “샘플 PDF 보기”.
  - Day 7: 면책·프라이버시 고지, 비용 모니터링, QA/배포. 초기 50명 피드백 루프.

- 기능 요약 (MVP)
  - Guided Capture + QC: 찍어야 할 각도 가이드, 불량샷 재촬영 유도.
  - Vision 주석/영역 추정: 누수 흔적/싱글 손상/타공 등 감지 및 하이라이트.
  - RAG 요약/체크리스트: 보험사 선택 시 템플릿 문구로 자동 정리.
  - Triage 제안: “고/중/저” → 파일링/모니터/시공사 연결.
  - 한 페이지 Claim Packet 내보내기(PDF/ZIP): 유료 핵심.

- 수익 모델(가설)
  - 미리보기 무료, 고해상도 비워터마크 PDF 내보내기 IAP: 4.99–9.99달러/건.
  - Pro 구독(홈오너/시공사): 월 9.99–19.99달러(무제한 내보내기, 브랜딩/템플릿, 고속 처리).
  - 선택: 시공사 리드 제휴(옵트인).

- 핵심 지표
  - 촬영→분석→PDF 미리보기 전환율(>40%), 미리보기→유료 내보내기 전환율(>8–12%).
  - 1세션당 절약 시간(목표 30분+), 시공사 견적 전환율 상승(+10%p).
  - OpenAI 호출/평가당 비용(<$0.20) 유지, 재방문률(30일 내 >25%).

- 리스크와 가드레일
  - 법적 책임: “전문가 검사 아님” 명시·확률/근거 표시, 유료 ‘휴먼 리뷰’ 옵션.
  - 보험사 포맷 편차: 상위 5개 보험사 템플릿부터 깊게 최적화, 나머지는 제너릭 포맷.
  - 비용 폭주: 온디바이스 QC로 불필요한 Vision 호출 차단, 유료 내보내기에만 풀 RAG.

- 초기 고투마켓
  - 폭풍/우박 지역 커뮤니티·페북 그룹·지역 시공사 파트너와 번들(첫 PDF 무료 쿠폰).
  - “보험사 문구 자동 생성” 전면 내세워 차별화. 사용 전/후 샘플 PDF 공개로 신뢰 확보.

결론: 사진 주석 + 보험사 맞춤 요약 + 즉시 내보내기 결합이 직접적인 시간·돈 절약으로 이어져 결제 유인이 강합니다. Supabase Vector/Auth와 OpenAI Vision/GPT로 7일 내 MVP가 가능한 구조이며, 유료 내보내기와 Pro 구독으로 즉시 매출 검증 가능합니다. 이 기능이면 유료 구독 쌉가능입니다! 💸
