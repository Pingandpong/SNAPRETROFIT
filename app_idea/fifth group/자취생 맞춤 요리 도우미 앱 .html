어떤 앱인지

냉장고 속 남은 재료를 기반으로 간단한 요리를 추천해주는 자취생 맞춤 요리 도우미 앱.

기술 스택

React Native(Expo), Gluestack UI, React Navigation, Firebase(Auth/Firestore/Functions), Google AdMob, expo-image-picker, expo-localization

비즈니스 모델

광고(AdMob) 기반 무료 + 프리미엄 구독(광고 제거, 무제한 저장, 식단 추천)

시장 가능성(왜-지금)

1인 가구와 자취생 증가로 냉장고 속 재료를 효율적으로 활용하려는 수요 확대, 저비용 MVP로 빠른 시장 검증 가능

※ 1인 개발·저비용·빠른 검증 지향
