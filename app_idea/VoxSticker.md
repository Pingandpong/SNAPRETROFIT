최종 추천 MVP: VoxSticker — “한 번 녹음 → 즉시 보이스 퍼소나 스티커 → 릴스/TikTok 업로드” (Whisperless 풀프로덕트로 확장 전, 가장 빠른 바이럴 루프)

왜 지금인가
- 트렌드: HF Spaces·GitHub에서 RVC/실시간 보이스 변환·MusicGen 데모가 연일 상위권. 크리에이터들은 “짧고 강렬한 오디오 밈”을 찾고 있고, 플랫폼은 출처 표기(워터마크·프로비넌스)를 요구 중.
- 공백: 모바일에서 “즉시 보이스 캐릭터 + 7초 공유 포맷 + 출처 인증”을 깔끔하게 붙인 제품이 없음.

타깃 유저
- 숏폼 크리에이터/뮤지션 지망생/밈 메이커. 하루 1–3회 업로드, 아이디어는 있는데 보컬·믹싱 리소스가 부족한 사용자.

핵심 사용자 여정 (30초)
- 7–10초 샘플 녹음 → ‘Create Persona’ 탭 → 즉시 미리보기(저품질 온디바이스) → 7초 스티커 자동 생성(파형 애니메/퍼소나명/프로비넌스 배지) → 릴스/TikTok 공유.
- 유료 클릭 포인트: 워터마크 제거, HQ 오디오/비트레이트 업스케일, 스템 분리, 상업용 라이선스 토큰.

Money-Making AI Feature
- 상업용 스템/HQ 익스포트 + 프로비넌스 토큰 번들
  - 유료 구독자는 HQ(24 kHz+) 오디오와 보컬/하모니 스템 분리, 그리고 “서명된 프로비넌스 토큰(출처·라이선스 포함)”까지 한 번에 구매/다운로드.
  - 왜 지불하나: 브랜드 협찬 영상·유튜브 수익화·음원 샘플 판매에 “출처 증명 + 스템”이 바로 필요. 이 번들로 편집·권리응답 대기·재녹음 시간을 주당 3–5시간 절감.
  - 추가 수익: 1클립 상업 라이선스 IAP(예: 4.99달러), 스템 익스포트 크레딧 팩, 프리미엄 퍼소나/하모니 프리셋 팩.

수익 모델(초기 권장)
- 프리미엄 구독: 월 6.99달러(워터마크 제거, HQ 익스포트, 기본 스템), 프로 19.99달러(우선 렌더 + 상업용 토큰 포함).
- IAP: 1클립 상업 라이선스, 스템/HQ 익스포트 크레딧, 퍼소나·하모니 스타일 팩.
- 리워디드 광고: 무료 유저의 추가 클라우드 렌더 크레딧 확보용으로 최소 도입.

MVP 스코프(7일 빌드 플랜, 빠른 바이럴 우선)
- Day1–2: Expo Router+NativeWind 셋업, Supabase Auth/Storage, 기본 녹음 및 업로드. HF Inference로 클라우드 보이스 변환 PoC.
- Day3–4: 온디바이스 저용량 변환(저품질 미리보기) 파이프라인, 파형 애니메이션/퍼소나 네이밍/간략 동의 체크(Consent) UI.
- Day5: 7초 MP4 스티커 익스포트(가시 워터마크 + 내장 메타데이터). 프로비넌스 토큰(서명 JSON) 생성·임베딩.
- Day6: 릴스/TikTok 공유 인텐트, 기본 분석/리퍼럴 훅. 유료 결제 게이트(플랫폼 IAP)와 Supabase 연동.
- Day7: 안전장치(유명인 매칭 키워드 자동 플래그), QA, 내부 베타 론치.

기술 설계 하이라이트
- 시간 절약 포인트 1 — Supabase Auth: 소셜 로그인/세션/권한 분기를 서버 없이 신속 구축. 초기 계정·구독 연동 작업 시간을 수주 → 수일로 단축.
- 시간 절약 포인트 2 — Supabase Vector: 
  - 퍼소나 임베딩·프리셋 유사도 검색(가장 잘 맞는 톤/하모니 추천).
  - 간단한 표절/유사도 프리스크리닝(‘상업용 익스포트 허용’ 전 자동 체크)로 운영 비용·리스크 감소.
- 시간 절약 포인트 3 — OpenAI Vision:
  - 가사/스크립트 스크린샷 자동 파싱 → 타이밍 가이드/발음 힌트 생성.
  - 썸네일/표지 이미지에서 무드 추출 → 추천 하모니/이펙트 즉시 제안.
  - 결과: 대본 정리·타이밍 맞춤 등 반복 편집 시간을 주당 2–3시간 절감.
- AI 추론: 
  - 온디바이스(저용량 RVC 변형)로 즉시 미리보기 → 비용 제로/지연 최소.
  - HQ/스템은 Hugging Face Inference(또는 OpenAI Audio) 하이브리드 렌더 큐.
- 프라이버시·정책 대응:
  - 디폴트 온디바이스 미리보기, 동의 체크, 가시 워터마크, 내장 프로비넌스 메타.
  - 상업용/마켓플레이스 전 ‘플래그→수동 검수’ 파이프라인.

사람들이 왜 돈을 내는가
- 즉시 쓸만한 보이스 캐릭터 + 짧은 스티커 포맷은 조회수 드라이버. HQ/스템/라이선스가 붙으면 “협찬·수익화·음원 판매”가 가능해짐.
- 워크플로우 절감:
  - 보컬 녹음/리테이크/믹싱 커뮤니케이션 왕복 3–5시간/주 → 버튼 몇 번으로 대체.
  - 대본·타이밍 준비 2–3시간/주 → Vision 보조로 자동화.
  - 합계 5–8시간/주 절감. 월 20–30시간을 6.99–19.99달러로 사는 셈. 체감 ROI가 명확해서 지불 장벽이 낮음.

핵심 KPI
- D7 Retention ≥ 25% (온디바이스 즉시 미리보기 덕)
- 7초 스티커 공유 비율 ≥ 40% (첫 세션)
- 무료→유료 전환 ≥ 4% (HQ/스템/상업 토큰 게이트)
- 렌더당 클라우드 COGS ≤ 0.15달러(평균 7초 기준)

리스크와 완화
- 유명인 음성 모사 남용: 동의 플로우, 키워드·임베딩 유사도 자동 플래그, 상업용 허용 전 수동 검수, 가시 워터마크·프로비넌스 강제.
- 인퍼런스 비용 폭증: 온디바이스 미리보기 기본, 무료 크레딧 상한, 리워디드 광고/유료 크레딧.
- 앱스토어 정책: 프라이버시 우선(로컬 처리 기본), 명확한 동의/워터마크, 신고/차단 루프를 초기부터 탑재.

로드맵(3주 내 확장)
- 주2: 유료 번들(“HQ+스템+상업 토큰”) 출시, 추천 하모니·톤 프리셋을 Vector Search로 개인화.
- 주3: 미니 마켓플레이스(인바이트 온리) 오픈, 크리에이터 프리셋 팩 수수료 모델 도입.
