# Investor One-Pager Builder (투자자용 1페이지 자동 생성기)

## 1. 제품 개요
**대상:** 초기 스타트업 창업자  
**문제:** 투자자에게 보낼 깔끔한 1페이지 소개자료(PDF)를 만들기 어렵다.  
**해결:** 창업자가 핵심만 입력하면, 투자자 친화적 톤으로 다듬고 예쁜 PDF로 자동 변환.  
**핵심 가치:** 빠름(즉시), 보기 좋음(투자자용 톤/레이아웃), 신뢰 가능(증거 링크 포함)

---

## 2. 핵심 기능 (MVP 범위)

### ① 단일 입력 폼
- Problem (문제)  
- Solution (해결)  
- Traction / Metrics (핵심 지표)  
- Team  
- Ask (투자 요청: 금액/용도)  
- Proof Links (성과·후기·지표 스크린샷 URL)

### ② 카피 폴리싱
- 투자자 친화 문장으로 자동 리라이팅  
- 1~3가지 버전 제안  
  - 공격적 성장 강조형  
  - 데이터 중심형  
  - 보수적 신뢰형  
- 창업자가 직접 수정 가능

### ③ PDF 템플릿 출력
- 템플릿 1~2종 (단순·안정형)
- 공통 섹션: 로고 / 문제 / 해결 / 지표 / 팀 / Ask  
- 색상·폰트 최소 커스텀  
- Proof Links 하단 삽입 (신뢰 확보)

### ④ 결과물 배포
- PDF 다운로드  
- 공유용 토큰 URL 자동 생성 (읽기 전용 뷰 제공)

### ⑤ 컴플라이언스/신뢰 장치
- 문서 하단에 면책문구 자동 삽입  
- Proof Links 비어있을 경우 경고 표시

## 3. 아키텍처 / 기술 스택

| 구분 | 사용 기술 | 설명 |
|------|------------|------|
| 프론트엔드 | React Native + Expo | 웹/모바일 겸용, 빠른 빌드 |
| 백엔드/DB | Firebase (Firestore, Hosting, Auth) | 데이터 저장, 링크 호스팅 |
| 서버리스 실행 | Cloud Run | OpenAI 호출 중계 / PDF 변환 |
| AI | OpenAI API | 카피 다듬기 / 문체 변환 |
| PDF 변환 | HTML → PDF | 안정성 우선, 템플릿 1~2개만 |

**비용 절감 포인트**
- 캐싱된 결과 재사용  
- 요청 시만 Cloud Run 실행  
- 최소 템플릿 운영

## 6. 왜 지금인가

- 초기 투자 라운드는 **속도**가 중요  
- 슬라이드 12장보다 **한 장짜리 요약**이 훨씬 전달력 높음  
- 창업자는 디자인·카피에 약하고, 시간도 부족함  
- 본 제품은 이를 세 가지로 해결:
  1. 구조: 섹션 순서 고정
  2. 카피: 투자자 톤 자동 변환
  3. 비주얼: PDF 완성본 자동 생성  

**→ “초기 창업자의 첫 투자자용 문서”를 자동화하는 제품**

---

## 7. 개발 단계별 우선순위

1. Firestore 스키마 정의 (입력값·버전 관리)  
2. Expo 폼 화면 + 프리뷰 구성  
3. Cloud Run API (OpenAI 호출 → 카피 버전 생성)  
4. 선택 버전으로 A4 HTML 레이아웃 생성  
5. HTML → PDF 변환 API 구성  
6. Firebase Storage에 업로드 + 공유 링크 발급
