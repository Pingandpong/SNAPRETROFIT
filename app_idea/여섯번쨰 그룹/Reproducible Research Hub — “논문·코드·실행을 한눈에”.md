# Reproducible Research Hub — “논문·코드·실행을 한눈에”

---

## 📘 앱 개념 & 핵심 목표
최신 논문과 GitHub 코드를 한눈에 보고, “바로 실행 가능한 로컬 가이드”를 자동으로 만들어주는 앱.  
연구자나 소규모 팀이 새 모델을 빠르게 따라 해보고 비교·정리하는 시간을 획기적으로 줄이는 것이 목표.

---

## ⚙️ 간소화된 기술 스택
- **프런트엔드:** React Native (Expo) — 모바일/웹 공통 UI  
- **백엔드:** Firebase (Auth, Firestore) — 로그인 및 피드/북마크 저장  
- **서버:** Cloud Run — 논문/리포 메타데이터 처리(요약·크롤링), 무부하 시 0원 유지  
- **AI:** OpenAI API — 논문 요약, 키워드 매칭(초기 검증용, 사용량 제한 운영)

---

## 🚀 MVP 핵심 기능
1. **개인화 논문·리포 피드** — 관심 주제/키워드 기반 자동 추천  
2. **GitHub 자동 연결** — README 핵심 요약 표시  
3. **원클릭 로컬 실행 가이드** — 패키지·명령어 자동 정리, 실행 스크립트 다운로드  
4. **북마크 및 알림 기능** — 새 논문/리포 업데이트 시 푸시 또는 이메일  

> 💡 *초기에는 호스티드 샌드박스 실행(원격 실행)은 제외 — 로컬 중심으로 비용 절감*

---

## 💰 개발 및 운영 비용
- **총개발비 (1인 기준):** 약 ₩2,000,000  
- **월 유지비:** 약 ₩70,000  
  - Firebase: 무료/저요금 구간  
  - Cloud Run: 요청 없을 땐 0원  
  - OpenAI API: ₩30,000~₩50,000 수준 (요약/매칭 최소화 운영)

---

## ⚠️ 리스크 & 유의사항
- 리포별 실행 방법이 달라 자동 스크립트 정확도는 제한적 → 초기엔 수동 규칙 보완 필요  
- OpenAI API 비용 급증 방지 위해 **캐시·호출 제한** 필수  
- 일부 코드/데이터 재배포 금지 가능성 → **링크·요약 중심 저장**으로 회피  
- 원격 실행 기능은 비용 급등 가능 → **유료 크레딧 출시 전까지 보류**

초기엔 요약·가이드 자동화 정확도와 사용자 피드백 확보에 집중 💪
