# 🧩 프로젝트 기능 요약 --- IndieAI Studio

## 📱 앱 개념 & 핵심 목표

-   React Native(Expo) 기반의 **모바일 AI 기능 키트(SDK) + 샘플 앱**
-   **온디바이스 경량 멀티모달 모델(이미지+텍스트)** 로 즉시 동작
-   무거운 작업만 선택적으로 **클라우드 폴백(OpenAI/Cloud Run)**
-   목표: 인디·소규모 팀이 하루 만에 **캡셔닝/스타일 필터** 같은 AI
    기능을 앱에 쉽게 붙이도록 지원

------------------------------------------------------------------------

## 🧠 기술 스택

-   **프런트:** React Native / Expo (빠른 개발·배포)
-   **백엔드:** Firebase (Auth / Firestore / Analytics --- 서버리스
    사용자 관리)
-   **서버리스:** Cloud Run (무거운 추론용, 무부하 시 0원)
-   **모델:** TFLite / Core ML / ONNX 기반 경량 온디바이스 모델
-   **AI API:** OpenAI API (선택적 폴백 / 고난도 작업 전용)

------------------------------------------------------------------------

## 🚀 MVP 핵심 구성 (1인 개발 기준)

1.  **경량 RN SDK** --- `loadModel()`, `infer(image, text)` 단순 API
    제공
2.  **샘플 앱 2종**
    -   이미지 캡셔너
    -   스타일 필터 (카메라 연동, 오프라인 우선)
3.  **무료 티어 쿼터 시스템** --- Firebase로 사용량 카운트 + 업그레이드
    안내
4.  **Cloud Run 폴백 엔드포인트 1개** (기본은 온디바이스 우선)
5.  **퀵스타트 가이드** --- "10분 안에 첫 결과"를 목표

------------------------------------------------------------------------

## 💰 예상 비용 구조

-   **초기 개발비:** 약 ₩1,000,000 (도메인·계정·테스트 단말 포함, 인건비
    제외)
-   **월 유지비:** ₩30,000 \~ ₩80,000
    -   Firebase 무료/저요금제
    -   Cloud Run 무부하 시 0원
    -   OpenAI API는 사용량 기반
