# 2025 10 16 - 최소비용 운동 앱 MVP 제안

---

## ① 어떤 앱인지  
**“챗 기반 퍼스널 코치” 운동 앱**  
사용자는 간단한 설문으로 목표를 입력하면, AI 코치가 매일 맞춤 루틴과 팁을 대화 형식으로 제안합니다.  
핵심 목표는 **① 운동 결정을 쉽게**, **② 습관을 만들도록 돕는 것**입니다.

---

## ② 기술 스택  
- **프론트:** React Native + Expo (iOS/Android 동시 개발)  
- **백엔드:** Firebase (Auth / Firestore / Storage)  
- **서버:** Cloud Run (소형 엔드포인트, 웹훅·LLM 프록시 등)  
- **AI:** OpenAI API (코칭 답변 및 요약 생성)  
- **비디오:** 외부 유튜브 비공개 링크 활용 — 스토리지/대역폭 절감  

---

## ③ 비즈니스 모델  
- 초기에는 **무료 MVP**로 출시 → 사용자 습관 형성 및 반응 검증  
- 2차 릴리즈에서 **구독 모델** 또는 **프리미엄 플랜** 도입  
- **월 유지비 목표:** ₩70,000 이하  
  - Firebase 무료 구간  
  - OpenAI 호출량 제한 / 캐시 / 프롬프트 템플릿 활용  

---

## ④ 시장 가능성 (왜 지금?)  
홈트레이닝 앱 시장은 이미 포화지만 대부분 **영상 중심**으로 복잡하고 무겁습니다.  
이 앱은 **“초간단 데일리 루틴 + 습관 스틱 + 가벼운 장비 기반”** 접근으로,  
**AI 대화 기반의 가벼운 경험**을 제공하여 차별화할 수 있습니다.

---

## 💰 예상비용  
- **개발비:** 약 ₩2,000,000 (1인 개발, 2~3주)  
- **월 유지비:** 약 ₩70,000 이하

## ⚠️ 리스크 & 대응  
- **AI 코칭 부정확성:** 명확한 면책 문구, 보수적 가이드 제공  
- **비용 급증 위험:** 일일 호출 캡, 캐시, 프리셋 답변 활용  
- **개인정보 이슈:** 최소 수집, 명확한 동의 정책 적용
