# RapidTopic — 원클릭 시장 스캔  

### 📘 어떤 앱인지  
사용자가 **한 줄 주제만 입력**하면,  
48–72시간 내에 핵심 트렌드 · 경쟁사 스냅샷 · 다음 실험 아이디어 3가지를 담은  
**1장짜리 시장 리포트**를 자동 생성.  

> “이 시장, 내가 지금 들어가도 되는가?”를 빠르게 판단할 수 있게 함.  

**리포트 구성:**  
- 🔹 트렌드 Top 3 — 지금 시장을 움직이는 흐름  
- 🔹 경쟁/대안 Top 3 — 이미 수익 내는 플레이어  
- 🔹 다음 실험 3가지 — 바로 실행 가능한 테스트 아이디어  

---

### ⚙️ 기술 스택  
| 구성 | 기술 | 설명 |
|------|------|------|
| 프런트엔드 | **React Native / Expo** | 주제 입력, 리포트 조회 UI |
| 백엔드 | **Firebase (Auth / Firestore)** | 사용자·요청·리포트 저장 |
| 서버리스 | **Cloud Run** | 뉴스·커뮤니티·GitHub 데이터 수집 및 요약 |
| AI | **OpenAI API** | 시장 요약, 경쟁사 분석, 실행 제안 자동화 |
| 출력 | **PDF Generator** | 깔끔한 1장 리포트 생성 |

> 서버리스 기반으로 유지비 최소화, 1인 운영 가능 구조.

---

### 💰 비즈니스 모델  
- 💵 **건당 결제 (메인)** — 첫 리포트 무료 체험, 이후 단건 구매  
- 🧩 **마이크로 업셀** — “같은 시장 내 후속 아이디어 3개 더” 등  
- 🔄 **구독 확장** — 특정 시장 주제의 지속 알림형 리포트  

> 초기부터 “무료 트래픽”이 아닌 **직접 결제형 구조**로 설계.

---

### 🚀 시장 가능성 (Why Now)  
- 📉 **시장 리서치 비용이 너무 비쌈** — 수주일·수십만 원 단위 → RapidTopic은 하루 만에 핵심만  
- 🤖 **AI가 노이즈를 ‘맥락’으로 바꿔줌** — 뉴스·토픽을 실질적 인사이트로 변환  
- 👤 **1인 창업·린 팀 급증** — “아이디어 검증용 미니 리서치” 수요 폭발  

> **이제는 아이디어 단계에서도 고급 시장 브리핑을 살 수 있는 시대.**

**Monetization:** 건당 결제 → 구독 확장
