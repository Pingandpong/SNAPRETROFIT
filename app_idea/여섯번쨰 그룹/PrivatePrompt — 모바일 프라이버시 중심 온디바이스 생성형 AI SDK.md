# PrivatePrompt — 모바일 프라이버시 중심 온디바이스 생성형 AI SDK

---

## 📱 앱 개념 & 핵심 목표  
React Native/Expo 기반으로 **“온디바이스 우선” 텍스트 생성/요약 SDK**를 제공하는 개발자용 도구.  

- 기본적으로 **기기 내 모델**로 빠르고 안전하게 추론  
- 길거나 복잡한 요청만 **자동으로 Cloud Run으로 폴백**  
- **개인정보 보호 중심 설계**(동의·오프라인 모드 포함)  
- 즉시 사용 가능한 **API(`prompt()`, `summarize()`)** 및 UI 컴포넌트 제공  

---

## ⚙️ 간단한 기술 스택  
- **RN / Expo** — 크로스플랫폼 샘플 앱  
- **온디바이스 모델** — 경량 LLM (quantized, llama.cpp / TF Lite)  
- **Firebase** — Auth + Firestore(권한·텔레메트리) + Storage(모델 파일 관리)  
- **Cloud Run** — 폴백 추론용 엔드포인트 (자동 스케일, 유휴 시 0원)  
- **OpenAI API (선택적)** — 초기 파일럿 단계에서만 제한적으로 사용  

---

## 🚀 MVP 1주 스프린트 목표  
1. **온디바이스 요약/답변 모델 연결** (기본 프롬프트 파이프라인 구축)  
2. **하이브리드 라우팅**: “로컬 우선 → 실패/장문 시 Cloud Run 호출”  
3. **프라이버시 UI**: 데이터 업로드 명시·오프라인 모드 토글  
4. **SDK API 구현**: `prompt()`, `summarize()` + RN 예제 앱  
5. **경량 텔레메트리**: 사용량·지연시간만 기록(개인정보 미전송)  

---

## 💰 예상 개발 비용 (1인 기준)  
- 총 개발비: **약 2,500만 원 (6–8주, 프로토타입~파일럿 수준)**  
- 월 유지비(목표 10만 원 이하)  
  - Firebase: 0–2만 원  
  - Cloud Run: 0–5만 원 (스케일=0 설계)  
  - OpenAI API(파일럿 한정): 0–3만 원  
  - **합계:** 0–10만 원 내 유지 가능  

---

## ⚠️ 잠재 리스크 및 대응  
- **모델 품질 격차**: 온디바이스 모델은 장문·복잡 입력에 약 → UX로 “고급 모드=클라우드” 표시  
- **앱 용량·배터리 이슈**: 모델 크기 튜닝 및 저사양 자동 폴백  
- **클라우드 비용 급증 방지**: 하드쿼터·일일 예산·캐싱 적용  
- **프라이버시 규제 대응**: 아동/교육용 대비 동의·삭제·내보내기 플로우 포함  

---

**요약:**  
PrivatePrompt는 “프라이버시 우선 + 온디바이스 중심” 철학으로 설계된 차세대 모바일 AI SDK입니다.  
로컬에서 대부분의 생성 작업을 수행해 **개인정보를 보호**하면서도, 필요한 경우만 **클라우드로 폴백**하는 효율적 구조를 목표로 합니다.
