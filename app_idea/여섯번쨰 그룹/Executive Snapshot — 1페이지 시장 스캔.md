# Executive Snapshot — 1페이지 시장 스캔

---

## 🧩 앱 개념 / 핵심 목표
- 사용자가 주제만 입력하면  
  **트렌드 3개 · 경쟁사 3개 · 성장 신호 3개 · 다음 액션 2개**를 자동 요약해주는 **“임원용 1페이지 리서치 요약 앱”**
- 회의 전 빠르게 읽고 공유할 수 있는 **실행 요약 보고서** 형태로 제공

---

## 🛠 기술 스택
- **React Native + Expo** → 모바일 & 웹 동시 개발  
- **Firebase (Auth / Firestore / Hosting)** → 로그인, 데이터 저장, 서버리스 배포  
- **Cloud Run (호출형, 경량)** → LLM 호출 및 PDF(옵션) 처리  
- **OpenAI API** → 요약 및 보고서 구성 생성

---

## 🚀 MVP 핵심 기능
1. 주제 입력 + 기간(30/60/90일) + 지역 선택 + “기본값으로 진행”  
2. LLM 기반 1페이지 초안 자동 생성 (웹 크롤링·인용 제외)  
3. 섹션별 간단 편집 및 저장 (히스토리 1단계)  
4. 공유 링크 생성 + 클라이언트 측 PDF/이미지 내보내기  
5. 결과 캐싱 및 토큰 제한으로 API 비용 절감

---

## 💰 개발비 & 유지비
- **예상 개발비:** 약 **₩16,000,000** (1인 개발 2~3주 + 베타 테스트 포함)  
- **월 유지비:** 약 **₩70,000 ~ ₩90,000**
  - Firebase 무료/저가 구간 활용  
  - Cloud Run 스케일-투-제로  
  - OpenAI 사용량 상한(₩50,000)  
  - 캐싱 적용

---

## ⚠️ 리스크 / 주의사항
- **LLM 환각:** 인용 없는 요약 → “검토 필요” 배지 표시  
- **비용 폭증 방지:** API 상한·캐싱·길이 제한  
- **PDF 품질:** 초기엔 클라이언트 내보내기만 제공  
- **개인정보 보호:** 비공개 저장 옵션 및 삭제 기능 필수
