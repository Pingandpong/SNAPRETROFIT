추천 MVP: SoundScape — One‑Tap Ambient Remix (카메라+마이크로 6~12초 촬영 → 즉시 루프형 앰비언트 트랙 3종 미리듣기 → 원탭 소셜 내보내기)

- 왜 지금: MusicGen·RVC·Whisper 계열이 지난 30일 트렌딩. 짧은 리믹스/배경음이 틱톡·릴스에서 바이럴 견인. 모바일에서 “즉시 프리뷰 → 유료 HQ 렌더” 모델이 수익/성능 균형에 적합.

- 대상 유저
  - 숏폼 크리에이터: 저작권 안전한 맞춤 배경음이 필요, 1분 내 결과 선호.
  - 집중·공부 사용자: 카페 소음+잔향 유지한 프라이빗 포커스 사운드.
  - 아마추어 뮤지션: 10초 보컬/기타 스케치 → 데모 루프/스템.

- 핵심 가치
  - 음악 라이브러리 뒤지기/편집 30~40분 → 1분 이내 원탭 생성. 주 5~6개 클립 기준 최소 주 3시간 절감.
  - 저작권/뮤트 리스크 감소, 일관된 톤&무드로 조회수/완주율 상승.
  - 프라이버시 우선: 기기 내 즉시 프리뷰, 필요시에만 클라우드 HQ 렌더.

- Money-Making AI Feature
  - 유료 HQ Cloud Render + Stem Export: 로컬 미리듣기(8~12초, 저지연) 후, 결제로 30~90초 고음질+스테레오/무손실 렌더와 드럼/베이스/패드 스템 번들 제공.
  - 옵션: 프라이빗 보이스 텍스처(RVC 스타일) 저장은 구독 전용. “장면→무드 자동 매칭(비전)” 기반 프리셋 추천은 무료 티저로 전환율 상승.

- 코어 흐름 (1분)
  1) 원탭 캡처(3~12초 영상+환경음)
  2) OpenAI Vision로 프레임 분석 → ‘Dreamy/Lo‑fi/Seaside’ 무드 추천
  3) 기기 내 양자화 MusicGen 미리듣기 3종 생성(8~12초 루프)
  4) 루프 트림/크로스페이드 → 소셜 내보내기
  5) 필요 시 유료 HQ 렌더/스템 내보내기

- 왜 돈을 쓰는가
  - 시간 절약: 주 3시간 아낀다면, 월 $7는 가성비. 크리에이터는 시간=업로드 빈도=수익.
  - 품질·일관성: 플랫폼 뮤트/저작권 경고 회피로 재업로드/편집 시간 손실 제로.
  - 차별화: 장면에 맞춘 독자적 사운드로 영상 완주율·저장률 상승(실제 전환 KPI로 설계).

- 기술 스택(출시 속도 최적화)
  - Frontend: Expo Router + NativeWind (빠른 네이티브 UI 구축)
  - Backend: Supabase Auth/Storage/Functions
    - Supabase Auth: 이메일/소셜 로그인, 비공개 보이스 텍스처 접근 제어.
    - Supabase Vector: 프리셋/무드 임베딩 저장 → 유사 무드 검색, 개인 취향 학습으로 추천 정확도↑. 프리셋/사용 내역 세션 RAG에 즉시 활용.
  - AI:
    - On‑device: 양자화 MusicGen류 ONNX/TFLite로 초저지연 프리뷰.
    - Cloud: Hugging Face Inference 또는 OpenAI Audio/Music API로 HQ 렌더·스템.
    - OpenAI Vision: 단일 프레임 분석→무드/템포/악기 프롬프트 자동 생성(프리셋 매핑 시간 절감).
  - 이것들이 시간을 어떻게 줄이나
    - Supabase Vector/Auth: 개인 취향/프리셋을 벡터화·권한관리까지 통합, 맞춤 추천·히스토리 RAG를 별도 인프라 없이 구현 → 백엔드 구축/운영 주 단위 절감.
    - OpenAI Vision: 장면 이해→프롬프트 자동화로 사용자 선택 과부하 제거 → 1~2초 내 무드 제안, 편집 단계 대폭 축소.

- MVP 범위(7일 빌드 플랜)
  - Day 1: 캡처(카메라/마이크) + Supabase Auth/Storage 연결
  - Day 2: 로컬 미리듣기 엔진(양자화 MusicGen) + 3변형 UI
  - Day 3: 간단 루프 편집기(트림/크로스페이드) + OpenAI Vision 무드 추천
  - Day 4: 경량 노이즈 클린(Whisper 파생 전처리)
  - Day 5: Supabase Functions 큐 + HF Inference로 30~90초 HQ 렌더(유료 게이트)
  - Day 6: 소셜 내보내기/쉐어카드 + 기본 분석(지표: 생성→미리듣기→내보내기→결제 전환)
  - Day 7: 프라이버시·동의 UX(보이스 텍스처 옵트인/삭제) + 테스터 배포
