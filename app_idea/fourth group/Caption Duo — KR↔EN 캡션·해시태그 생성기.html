어떤 앱인지 (한 줄)

  • 한 줄 키워드만 넣으면 KR/EN 캡션 2안과 해시태그 10개를 즉시 생성, 원탭 복사·공유까지.

기술 스택 (핵심)

  • Expo: expo-localization, expo-clipboard, expo-haptics
  • UI: Gluestack UI — Input/Select/Card/Toast
  • Firebase: Analytics(Generate/Copy/Share), Remote Config(A/B), Functions(경량 LLM 프록시)
  • MCP: 프롬프트/톤 프리셋·금칙어 사전 외부화, 실패 시 로컬 템플릿 대체

비즈니스 모델

  • 무료 + 광고(배너 + 생성 후 인터스티셜 1회/세션)
  • IAP: 광고 제거 + 프리셋 추가 + 해시태그 세트 저장(일회성)

시장 가능성 (왜-지금)

  • 숏폼/UGC 확대 → 짧은 캡션·해시태그 수요↑, 한/영 동시는 여전히 공백.
  • 짧은 호출·로컬 사전으로 비용 낮게 유지, 1주 내 수익 검증 가능.
