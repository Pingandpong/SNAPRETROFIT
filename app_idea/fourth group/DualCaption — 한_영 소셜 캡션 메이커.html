DualCaption — 한/영 소셜 캡션 메이커

왜-지금
  • 숏폼·UGC 증가로 짧은 캡션 수요 급증.
  • 한/영 동시 타겟 니즈 뚜렷 — 양방향 캡션 차별화.
  • 짧은 프롬프트 기반 LLM로 1인·저비용 검증 가능.
타겟 페르소나
  • 대학생·취준생: 자연스러운 영어 캡션 필요.
  • 소상공인: 홍보용 문구·해시태그 빠른 생성.
  • 해외 K-컬처 팬: 한국어 감성의 영문 캡션.
핵심 JTBD
  • 한 줄 → 5초 내 KR/EN 캡션 3안 + 해시태그.
  • 장문 → 짧/중/길 3버전 자동 리라이트.
  • 원탭 복사/공유 & 최근 결과 재사용.
최소 기능 (3–5)
  • 캡션 생성: 톤 선택 후 KR/EN 3안.
  • 해시태그 제안: 5–10개, 길이 옵션.
  • 복사·공유 & 최근 10개 저장, 즐겨찾기.
  • 템플릿 프리셋 3개(소개/제품/일상).
  • (옵션) 이미지 참조 기반 키워드 가중치.
기술 스택 & 적합성
  • Expo: expo-router, localization, clipboard, (image-picker 옵션).
  • UI: Gluestack UI (@/components/ui/... import 규칙).
  • 상태/검증: react-hook-form + zod.
  • Firebase: Analytics, Remote Config, Firestore, Crashlytics.
  • Ads: AdMob + iOS ATT(expo-tracking-transparency).
  • MCP: caption_templates/hashtag_suggest/guardrails 리소스화.
수익모델
  • Free + 광고(배너/복사 후 인터스티셜 1회/세션).
  • Pro(IAP/구독): 광고 제거, 톤 20+, 템플릿 커스텀, 무제한 저장.
  • 예시 가격: 월 ₩2,500 / 연 ₩19,000 또는 일회성 ₩5,500.
7일 검증 플랜
  • D1–2: 입력→3안→복사/공유, 자동감지, 최근 저장.
  • D3–4: 톤 4종, 해시태그 길이, 톤 기본값 A/B.
  • D5: AdMob 연결, 이벤트 로깅.
  • D6–7: UAC 소액 테스트(KR/EN), 스토어 스크린샷 A/B.
  • 생성율 ≥ 60%, 복사/공유율 ≥ 40%, D1 ≥ 20%.
  • CPI: KR ≤ ₩1,200, EN ≤ ₩1,800.
  • 200 installs 후 생성율 < 40% 또는 복사/공유율 < 25% → 피벗/중단.
초기 마케팅
  • ASO(KR): 캡션, 문구, 인스타 문구, 해시태그, 글귀.
  • ASO(EN): caption generator, instagram captions, hashtags.
  • 스크린샷: 한/영 동시 생성 / 3안 제안 / 원탭 복사.
  • 광고 훅: Before→After, 니치(카페/제품), 해시태그 속도.
리스크 & 중단 기준
  • 레드오션 → 속도/양방향 집중으로 차별화.
  • LLM 비용 변동 → 오프라인 사전·템플릿로 호출 최소화.
  • 정책/민감 콘텐츠 → Guardrails 선적용.
  • 중단: 위 지표 하회 시 즉시 중단 또는 기능 단순화.
경쟁 앱 & 차별점
  • Canva: 디자인 강점 ↔ 본 앱은 초고속 텍스트 캡션 특화.
  • CapCut: 영상 편집/자막 ↔ 본 앱은 문구·해시태그 즉시 생성.
  • Captions (AI): 영상 중심 ↔ 소셜 캡션·해시태그 특화.
  • Copy.ai / Rytr: 장문 카피 ↔ 짧은 문구 3안·원탭 복사.
  • Notion AI: 범용 ↔ UGC 캡션 전용 템플릿·오프라인 사전.
