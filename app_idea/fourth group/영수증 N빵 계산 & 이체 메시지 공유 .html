SplitQuick — 영수증 N빵 계산 & 이체 메시지 공유

왜-지금
  • 더치페이 잦아짐, KR(부가세 포함)/글로벌(세금·팁 분리) 혼재.
  • 과기능 앱 대비 즉석 1회 계산 + 공유 수요.
  • KR/EN 자동 안내로 언어 장벽 최소화.
타겟 페르소나
  • 대학생·사회초년생: 회식/모임 N빵.
  • 팀 총무: 카톡/문자 공유 필수.
  • 여행 동행(혼합 국적): 팁·세금 가감 필요.
핵심 JTBD
  • 총액·인원 입력 → 10초 내 1인 금액 산출.
  • 카카오/문자/WhatsApp으로 KR/EN 안내 즉시 전송.
최소 기능 (3–5)
  • 빠른 분할(반올림/버림).
  • VAT 포함/미포함, 팁 % 슬라이더.
  • 예외 인원 제외, 가중치(1.5배).
  • 공유 템플릿(KR/EN) + 원탭 복사·공유.
  • 최근 그룹 3개 저장.
기술 스택 & 적합성
  • Expo: expo-router, expo-localization, expo-clipboard, expo-haptics, Share
  • 폼: react-hook-form + zod
  • UI: Gluestack UI (@/components/ui/...)
  • Firebase: Analytics, Remote Config, Crashlytics
  • Ads: AdMob(배너+인터스티셜 1회/세션)
  • IAP: 광고 제거/무제한 그룹/가중치 확장
  • MCP: split.rules / message.templates / currencies.meta
수익모델
  • 무료 + 광고(결과 화면 배너, 공유 후 인터스티셜 1회).
  • IAP(일회성 ₩3,900~₩5,500): 광고 제거 + 무제한 그룹 + 고급 가중치 + 커스텀 템플릿.
7일 검증 플랜
  • D1–2: 입력→계산→공유 템플릿 복사/공유.
  • D3–4: VAT/Tip, 반올림/버림, 예외/가중치, 최근 그룹.
  • D5: AdMob, Remote Config A/B(팁 기본값), Analytics.
  • D6–7: KR/EN 스토어 & UAC 소액, 스샷 A/B.
  • 온보딩→첫 계산 ≥ 75%, 계산→공유 ≥ 50%.
  • 평균 계산 시간 ≤ 30초, eCPM ≥ $2.0, ARPDAU ≥ $0.01.
  • 공유 < 30% 또는 ARPDAU < $0.005 지속 → 중단/피벗.
초기 마케팅
  • ASO(KR): N빵 계산기, 더치페이, 회식 정산, 영수증 분할.
  • ASO(EN): bill splitter, tip calculator, split bill.
  • 스크린샷: “10초 계산” / “VAT·Tip 토글” / “KR/EN 공유”.
  • 광고: Before→After(엑셀 vs 앱), 회식 영수증 → 결과 금액 애니메이션.
리스크 & 중단 기준
  • 실제 송금 연동 아님 → 텍스트 공유만 명확히 안내.
  • 지역별 팁/세금 차이 → 수동 슬라이더 + MCP 기본값.
  • OCR 요구는 후순위(V2) — 초기 수동 입력.
경쟁 앱 & 차별점
  • Splitwise: 장부/채무 관리 ↔ 본 앱은 1회성 즉석 계산+공유.
  • Tab: 항목별 OCR ↔ 수동 10초 + KR/EN 메시지로 속도 우위.
  • Settle Up: 지속 그룹 정산 ↔ N빵 신속 계산 특화.
  • Plates by Splitwise: 좌석 기반 UI ↔ 반올림/버림·팁 슬라이더로 학습 곡선↓.
  • GoDutch: 인도 시장 중심 ↔ 한국/글로벌 혼합 KR/EN 템플릿.
