CleanShot — 스크린샷 청소기

RN/ExpoFirebase1인 개발 적합

어떤 앱인지 (한 줄)

  • 스크린샷을 자동 탐지하고 빠른 필터(최근/긴스샷/중복)로 골라 일괄 삭제·보관하는 초경량 정리 앱.

기술 스택 (핵심)

  • Expo: expo-media-library, expo-file-system, expo-localization
  • UI: Gluestack UI — 썸네일 그리드/하단 고정 액션
  • Firebase: Analytics, Remote Config, Crashlytics
  • MCP: 스크린샷 판별 규칙(폴더/해상도/유사해시) 외부화 → 앱 업데이트 없이 조정

비즈니스 모델

  • 무료 + 광고(배너, 작업 완료 후 인터스티셜 1회 한정)
  • IAP(일회성): 광고 제거

시장 가능성 (왜-지금)

  • 저장공간 압박·정리 피로 증가, 기존 앱은 광고 과다/과기능 경향 → 초간단·빠른 정리 틈새.
  • 온디바이스 처리 중심으로 비용 거의 0, 1주 내 검증 가능.
