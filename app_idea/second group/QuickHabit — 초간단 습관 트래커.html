QuickHabit — 초간단 습관 트래커

RN/Expo + Firebase · KR/EN 동시 타겟 · 1인 개발 즉시 출시용 MVP

앱 개요 및 핵심 기능

  • 핵심 JTBD: “매일 1분 이내로 습관을 체크하고, 성취감을 시각적 피드백으로 얻고 싶다.”
  • 최소 기능:
    1. 습관 생성(텍스트 + 아이콘/색상).
    2. 1탭 체크/해제(오늘의 목록).
    3. 일간·주간 달성률 미니 차트(스파크라인/도넛).
    4. 로컬 알림(expo-notifications), 조용한 시간 설정.
    5. KR/EN 인터페이스 전환.

기술 스택

  • Expo/RN: expo-notifications (알림), expo-device, react-native-svg (간단 차트), react-hook-form
  • Firebase: Auth(익명), Firestore(습관/체크 기록), Analytics, Remote Config(A/B)
  • UI (Gluestack UI): Card 리스트, Button/Checkbox 체크, Progress/Badge 달성률
  • MCP 활용: 습관명/루틴 자동 제안(KR/EN), 동기부여 문구 생성, 스토어 ASO 카피 다국어화

수익 모델

  • 무료+광고: 히스토리 화면 하단 배너, 습관 최대 5개.
  • 구독(월 ₩2,900): 광고 제거, 무제한 습관, 테마/아이콘, 위젯(후속).
  • IAP: 테마/아이콘 팩 일회성 구매.

시장성 (왜 지금?)

  • 한국: 자기계발·다이어트·스터디 등 라이트 유저용 “빠른 체크” 니즈 증가.
  • 글로벌: 무거운 기능 대신 1탭 체크 선호 트렌드 → 경량 앱 기회.
  • 실행 용이: 단순 CRUD + 로컬 알림 중심으로 1주 내 MVP 가능.

주요 경쟁 앱 및 차별점

  • Habitica — 게임화 강하지만 무겁고 영어 비중 큼 → KR/EN 초경량·1탭 체크 특화
  • Productive — 고급 기능·구독 압박 → 무료 기본 충분, 광고는 히스토리에만
  • Everyday / Streaks — iOS 중심·유료 위주 → Expo 기반 iOS/Android 동시, 저가 구독
  • Loop Habit Tracker(Android) — 기능 풍부하나 UI 복잡 → 미니 차트·간결 UI에 집중
