QuickCaption AI - 60초 이중 자막 자동 생성

GEMINI.md 베이스 · RN/Expo + Firebase + Gluestack UI · 한/영 동시 타겟

앱 개요 및 핵심 기능

  • 핵심 JTBD: “핸드폰에서 60초 영상만 올리면 한·영 자막이 자동으로 붙고, 스타일만 고른 뒤 바로 공유하고 싶다.”
  • 최소 기능:
    1. 영상 입력(≤60s): 촬영/갤러리 가져오기.
    2. STT + 번역: 언어 자동감지 후 KR↔EN 자막 동시 생성.
    3. 자막 프리셋 3종(폰트/색/배경) + 위치 드래그.
    4. 미리보기 & 내보내기: SRT 파일, (옵션) 워터마크 MP4.
    5. (옵션) 하이라이트: 무음·강조 단어 기반 컷 제안.

기술 스택

  • Expo: expo-av(재생), expo-image-picker/expo-document-picker(입력).
  • STT: Whisper API(또는 경량 ASR).
  • 번역/정제: gpt-4o-mini(구어체 자연화, 문장부호/줄바꿈 보정).
  • 렌더: 초기 SRT 우선, 하드서브는 Cloud Run + FFmpeg 대기열 처리.
  • Firebase: Auth, Storage(원본/자막), Functions(파이프라인), Firestore(Job 상태), Analytics, Remote Config.
  • UI(Gluestack UI): Card(미리보기), Tabs(KR/EN), Slider(폰트), Badge(워터마크), Button(렌더/공유).
  • MCP: 자막 정리 룰(시간코드/줄바꿈), 업종별 톤 프리셋, 다국어 ASO 문구 자동 생성.

수익 모델
