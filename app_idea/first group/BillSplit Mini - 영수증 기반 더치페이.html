BillSplit Mini - 영수증 기반 더치페이

GEMINI.md 베이스 · RN/Expo + Firebase + Gluestack UI · 한/영 동시 타겟

앱 개요 및 핵심 기능

  • 핵심 JTBD: “영수증을 찍기만 하면 항목·가격을 자동 추출하고, 사람별 금액을 몇 초 만에 나눠 공유하고 싶다.”
  • 최소 기능:
    1. 영수증 사진 OCR → 항목·가격 자동 인식(통화/세금/봉사료 감지).
    2. 사람 배정 UI(터치 분배): 균등/비율/개별 모드, 항목을 A/B/C에 드래그.
    3. 정산 결과 생성: 사람별 금액·송금 메모 복사, 공유 링크(영/한 자동).
    4. 수수료/할인/배달비 분배 규칙 토글.
    5. (옵션) 정산 요청 리마인더(알림 1회).

기술 스택

  • RN/Expo: expo-image-picker(갤러리/카메라), 필요 시 react-native-vision-camera.
  • OCR: Google ML Kit Text Recognition v2(네이티브 모듈 래핑).
  • 상태/유틸: zustand, dayjs, react-hook-form, react-navigation.
  • 숫자/통화: 간단 유틸 또는 dinero.js.
  • Firebase: Auth(익명/구글), Firestore(정산 세션/항목), Dynamic Links(공유), Analytics/Remote Config.
  • Gluestack UI: Card(영수증 미리보기), Tabs(분배모드), Badge(사람 태그), Sheet(세부 설정).
  • MCP: OCR 텍스트 → 항목 JSON 정규화, 통화/세금 규칙 추론, 한/영 결과문 생성, ASO 카피 자동화.

수익 모델

  • 무료+광고: 결과 화면 작은 배너(정산 전면 광고는 지양).
  • 구독(월 $2.49): 광고 제거, 공유 링크 무기한 유지, 정산 리마인더, 멤버 프리셋.
  • (확장) 결제/송금 파트너 제휴(국가별 노출)로 커미션.

시장성 (왜 지금?)

  • 한국: 회식·카페·배달 더치페이 일상화, 카톡 정산/엑셀이 번거롭고 오류 많음.
  • 글로벌: 스플릿 앱은 존재하지만 영수증→자동 분배 중심의 초경량 제품은 드묾.
  • 1인 개발로도 가능한 OCR/번역 API 보급, Google Ads로 빠른 유입 테스트 가능.
