DualNote AI - 요약·번역·태깅 AI 노트

GEMINI.md 베이스 · RN/Expo + Firebase + Gluestack UI · 한/영 동시 타겟

앱 개요 및 핵심 기능

  • 핵심 JTBD: “말하거나 붙여넣기만 하면 자동 요약/번역/태그가 끝나서 메모를 바로 공유·검색하고 싶다.”
  • 최소 기능:
    1. 입력: 텍스트/음성(녹음) 입력.
    2. STT → 텍스트 변환(언어 자동감지).
    3. 요약 + 번역(KR↔EN) 및 카테고리 태그 자동 부여.
    4. 저장 & 검색: 히스토리/즐겨찾기/키워드 검색.
    5. (옵션) 공유: 클립보드/노션/메일/카톡.

기술 스택

  • Expo: expo-av(녹음/재생), expo-file-system, expo-sharing.
  • STT: Whisper API(짧은 클립 우선), 장문은 서버 대기열 처리.
  • 요약/번역: gpt-4o-mini (요약 프롬프트 템플릿 + 한/영 동시 산출).
  • Firebase: Auth(익명/Google), Firestore(메모/태그), Storage(음성 파일), Analytics, Remote Config(A/B), Crashlytics.
  • UI (Gluestack UI): Button(녹음/정지), Card(요약 결과), Badge(태그), Tabs(KR/EN), Input(검색).
  • MCP: 요약 톤/길이 프리셋 자동화, 태그 사전 업데이트, 다국어 ASO 카피 생성.

수익 모델

  • 무료 + 광고: 최근 메모 20개 제한, 결과 화면 하단 배너.
  • 구독(월 $2.99): 광고 제거, 무제한 저장, 장문 요약/고급 번역, 폴더/검색 고급.
  • IAP: 긴 회의 파일 변환 크레딧(1시간 단위).

시장성 (왜 지금?)

  • 한국: 회의/강의/스터디 음성 기록 증가, 사후 정리 부담 큼.
  • 글로벌: AI 요약 붐에도 모바일 개인용 경량 요약+번역 툴은 드묾.
  • 기술: Whisper/STT + 경량 LLM로 저비용·빠른 체감 MVP 구현 가능.
