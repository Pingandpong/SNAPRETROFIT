FlashPitch - 피치덱 요약·30,60초 영문 스크립트

GEMINI.md 베이스 · RN/Expo + Firebase + Gluestack UI · 한/영 동시 타겟

앱 개요 및 핵심 기능

  • 핵심 JTBD: “피치덱 PDF만 올리면 슬라이드 핵심 요약(한·영)과 30초/60초 영어 피칭 스크립트를 즉시 얻고, 공유/다운로드하고 싶다.”
  • 최소 기능:
    1. PDF 업로드 → 슬라이드 이미지 분리/텍스트 추출.
    2. 슬라이드별 핵심요약 생성(한/영), 불릿 카드로 출력.
    3. 30초/60초 피칭 스크립트 자동 생성(톤: 비즈/친근/냉정).
    4. 다운로드/공유(텍스트·클립보드·노션).
    5. (옵션) TTS 오디오와 자막(SRT) 생성.

기술 스택

  • RN/Expo: expo-document-picker 또는 파일 업로드, EAS 빌드.
  • Storage: Firebase Storage(PDF/이미지) + Functions(서버리스 처리).
  • 요약/스크립트: OpenAI gpt-4o-mini + (Vision) 슬라이드 요약 파이프라인.
  • UI: Gluestack UI — Card(슬라이드 카드), SegmentedControl(30/60초), Accordion(세부).
  • DB/분석: Firestore(히스토리), Analytics, Remote Config(A/B 유료 제한).
  • MCP: 슬라이드→JSON 구조화, 한/영 동시 텍스트 생성, ASO/스토어 카피 자동 생성.

수익 모델

  • 무료: 월 3회 요약 + 광고.
  • 구독(월 $4.99~9.99): 무제한, 스크립트 템플릿, TTS/자막, PDF 자동 문구 정제.
  • IAP: 1회 요약권, 추가 TTS 크레딧.

시장성 (왜 지금?)

  • 한국: 초기 창업자/개발자 창업 증가 → 영문 피칭 문서·발표 부담이 큼.
  • 글로벌: 액셀러레이터·데모데이 확산, 30–60초 숏폼 피치 콘텐츠 수요↑.
