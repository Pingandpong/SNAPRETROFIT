EatSplit — 팁/부가세/N빵 초간단 계산기

RN/Expo + Firebase · KR/EN 동시 타겟 · 1인 개발 즉시 출시용 MVP

앱 개요 및 핵심 기능

  • 핵심 JTBD: “총액과 인원만 넣으면 팁·세금·반올림까지 반영된 1인 금액을 즉시 보여주고, 공유 카드로 바로 보내고 싶다.”
  • 최소 기능:
    1. 총액·인원·팁%·세금% 입력 → 즉시 1인 금액 계산.
    2. 반올림 옵션(₩10/100, ¢/¢10) 및 개별 조정(특정 인원 ±금액).
    3. 공유 카드(KR/EN 병기) 이미지/텍스트 생성.
    4. 국가 프리셋(KR/US/EU 기본 팁·세금) 선택.
    5. (선택) 최근 5건 기록 저장.

기술 스택

  • Expo/RN: expo-localization (언어/지역), expo-haptics (계산 피드백), expo-sharing (카드 공유), @react-native-async-storage/async-storage (로컬 기록), react-hook-form, @tanstack/react-query
  • Firebase: Analytics(이탈 구간), Remote Config(기본 팁% A/B), Firestore(선택: 기록/프리셋 동기화), Auth(익명)
  • Gluestack UI: Input, Select, Button, Card, Sheet(개별 조정), Toast
  • MCP 활용: 국가/언어 감지로 팁·세금 프리셋 추천, 공유 문구 KR/EN 자동 생성, ASO 카피 자동화

수익 모델

  • 무료+광고: 결과 화면 하단 배너, 전면광고는 월 1–2회만 제한적.
  • 구독(월 ₩1,900): 광고 제거, 기록 무제한, 영수증 OCR(후속).
  • IAP: 국가별 프리셋/테마 팩(일회성).

시장성 (왜 지금?)

  • 한국: 더치페이 일상화 + 해외여행 증가 → 팁/세금 포함 N빵 즉시 계산 수요.
  • 글로벌: 기존 스플릿 앱은 무겁거나 가입/초대 필요. 원탭 계산→공유 카드 경량 틈새.
  • 실행 용이: 입력→계산→공유 중심으로 1주 내 MVP 구현 가능.

주요 경쟁 앱 및 차별점

  • Splitwise — 가입·그룹 관리 중심, 무거움 → 즉시 계산·원탭 공유 경량 UX.
  • Tab — 식당 영수증 분할 특화 → 국가 프리셋 + 간편 반올림으로 일반 모임/여행 최적화.
  • Settle Up — 장부 관리·복잡 플로우 → 1회성 계산에 포커스.
  • BillPin — 친구 초대/동기화 전제 → 무가입·로컬 우선으로 속도.
  • 카카오톡 계산 보조(비공식) — 국내 채팅방 한정 → KR/EN 병기 카드로 글로벌 공유 최적화.
