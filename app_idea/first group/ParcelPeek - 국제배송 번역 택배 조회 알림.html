ParcelPeek - 국제배송 번역 택배 조회 알림

GEMINI.md 베이스 · RN/Expo + Firebase + Gluestack UI · 한/영 동시 타겟

앱 개요 및 핵심 기능

  • 핵심 JTBD: “운송장 번호(또는 상태 스샷)를 넣으면, 택배사가 달라도 진행 상황을 표준 타임라인으로 보여주고 어려운 상태 메시지를 한·영 생활어로 설명해 주며 단계별 푸시 알림을 받고 싶다.”
  • 최소 기능:
    1. 입력: 운송장 붙여넣기 / 스크린샷 OCR / 바코드 스캔(expo-barcode-scanner).
    2. 상태 수집·표준화: CJ/롯데/한진/우체국/DHL/UPS/FedEx 등 선택 → 도착/통관/배송중/완료 스키마 매핑.
    3. 한·영 쉬운 설명: 약어/전문용어를 생활어로 재해석(예: In transit → 이동 중, 다음 허브로 이동).
    4. 푸시 알림: 통관 시작/지연, 배송 출발/완료 트리거.
    5. (옵션) 타임라인·ETA: 허브 경로와 간단 ETA(룰/회귀).

기술 스택

  • Expo: expo-barcode-scanner, expo-image-picker, expo-notifications, expo-clipboard.
  • 상태 조회: 1차는 사용자 택배사 선택 + 운송장 → Cloud Functions에서 공식 페이지 HTML 파싱 또는 서드파티 트래킹 API 어댑터.
  • OCR 폴백: 실패 시 스샷 OCR → 텍스트 파싱.
  • LLM/룰 하이브리드: 로컬 JSON 룰로 코드→표준상태 매핑 + gpt-4o-mini로 KR/EN 쉬운 문장 재작성.
  • Firebase: Auth(익명/Google), Firestore(트래킹/타임라인), Functions(크론·조회·번역), Storage(스샷), Analytics/Remote Config(A/B).
  • UI (Gluestack UI): Card(현재 상태), Badge(표준 상태), 커스텀 Timeline, Button(알림 토글), Sheet(택배사 선택).
  • MCP: 상태 텍스트→표준 JSON 변환 템플릿, KR/EN 설명 자동 생성, 다국어 ASO 카피 생성.

수익 모델

  • 무료+광고: 최대 3개 트래킹 저장, 상태 화면 하단 배너.
  • 구독(월 $1.99~2.99): 광고 제거, 무제한 트래킹, 우선 조회(짧은 폴링), 통관 지연 특수 알림.
  • IAP: 해외 특급 모니터링 24시간 패스(고빈도 체크).

시장성 (왜 지금?)

  • 한국: 해외직구·역직구·중고거래 증가 → 택배사/용어 제각각, 상태 이해 어려움.
  • 글로벌: 한국↔해외 배송 모두 다루는 양방향 KR/EN 상태 설명 니즈 상시.
  • 틈새: 기존 트래킹 앱은 영문 약어/직역 중심 → 생활어로 풀어주는 ‘설명형’ 수요.
