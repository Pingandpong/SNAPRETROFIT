# 🏠 SnapRetrofit - AI 기반 홈 에너지 진단 앱

> 사진 몇 장으로 집의 에너지 낭비를 찾고, 절감액과 정부 보조금까지 알려주는 스마트 홈 에너지 컨설턴트

## 📱 앱 개념 (쉽게 설명)

### 문제: 에너지 비용이 너무 비싸요! 😰

- 전기/가스 요금이 계속 오르는데, 어디서 새는지 모르겠어요
- 단열 공사? 창문 교체? 뭐부터 해야 할지 모르겠어요
- 정부 보조금이 있다는데, 어떻게 신청하는지 복잡해요
- 전문가 부르면 비용만 수십만원...

### 해결: SnapRetrofit이 2분 만에 답을 줍니다! ✨

**사용 방법 (정말 간단해요!)**

1. **📸 사진 3장만 찍으세요**
   - 다락문 (단열재 확인)
   - 창문 (이중창인지 단창인지)
   - 실외기 (에어컨/히터 모델 확인)

2. **🤖 AI가 자동으로 분석**
   - "이 집은 단열이 부족해요"
   - "창문이 단창이라 열이 많이 새요"
   - "실외기가 15년 됐네요, 전기를 많이 먹어요"

3. **💰 즉시 리포트 받기**
   - **우선순위**: "창문 교체를 먼저 하세요!"
   - **절감액**: "연간 120만원 절약 가능"
   - **투자 회수**: "2년이면 본전"
   - **보조금**: "정부에서 50만원 지원 (신청 링크 첨부)"

## 🛠 기술 스택 (개발자용)

### Frontend (모바일 앱)
```
📱 React Native + Expo
   └─ Expo Router (파일 기반 라우팅)
   └─ NativeWind (Tailwind CSS for React Native)
   └─ expo-camera (사진 촬영)
   └─ expo-image-picker (갤러리 선택)
```

### Backend (서버리스)
```
☁️ Supabase (Firebase 대체)
   ├─ Auth (이메일/소셜 로그인)
   ├─ Database (PostgreSQL)
   ├─ Storage (사진 저장)
   └─ Vector (RAG용 벡터 검색)
```

### AI/ML
```
🤖 OpenAI API
   ├─ Vision API (사진 분석)
   │  └─ 단열재 감지
   │  └─ 창문 타입 인식 (단창/이중창)
   │  └─ 실외기 모델/연식 OCR
   │
   └─ GPT-4o (리포트 생성)
      └─ 우선순위 결정
      └─ ROI 계산
      └─ 보조금 매칭 (RAG)
```

### RAG (Retrieval Augmented Generation)
```
🔍 Supabase Vector + OpenAI Embeddings
   └─ 지역별 정부 보조금 데이터
   └─ 에너지 효율 규정
   └─ 공사 단가 정보
   └─ 시공사 정보
```
